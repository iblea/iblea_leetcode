# 예제 테스트

### CPP

GTEST(googletest)
- `googletest/include`, `googletest/lib` 내 `googletest` 관련 동적 라이브러리 필요.

### PYTHON

unittest 또는 pytest 라이브러리 사용


# 테스트 방법

### CPP

- `fullcode.cpp` -> `./gtest.sh` 실행

### PYTHON

- `fullcode.py` -> `python3 fullcode.py` 실행 또는 실행 권한이 있을 경우 `./fullcode.py` (첫 라인 `#!/usr/bin/python3` 필요)
